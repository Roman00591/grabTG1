# MyGrab Bot — Инструкция по запуску

## 1. Установка зависимостей
```
pip install -r requirements.txt
```

## 2. Настройка config.json
Заполните все поля:
- `api_id` / `api_hash` — получить на https://my.telegram.org
- `bot_token` — получить у @BotFather в Telegram
- `my_id` — ваш Telegram ID (узнать у @userinfobot)
- `technical_channel_id` — ID канала для модерации (бот должен быть админом)

## 3. Создание сессии Telethon
```
python authorization.py
```
Введите номер телефона и код из Telegram. Создастся файл `myGrab.session`.

## 4. Запуск бота
```
python main.py
```

## 5. Первоначальная настройка через Telegram
1. Напишите боту `/start`
2. Добавьте канал-источник: `/add_source`
3. Добавьте канал-получатель: `/add_dest`
4. Создайте связку: `/add_mapping`
5. Готово — бот начнёт копировать сообщения!

## Команды бота
- `/status` — текущие настройки
- `/moderation` — вкл/выкл ручную модерацию
- `/copywriting` — вкл/выкл рерайт через GPT
- `/gpt_mode` — переключить внутренний/внешний GPT
- `/set_prompt` — изменить промпт для GPT
- `/whitelist` — белый список слов
- `/blacklist` — чёрный список слов
- `/delete_words` — слова для обрезки текста
- `/rss` — управление RSS-лентами
- `/add_admin` — добавить администратора

## Примечания
- Без модерации: сообщения пересылаются автоматически
- С модерацией: каждое сообщение приходит в технический канал с кнопками
- GPT внутренний (g4f) — бесплатно, но нестабильно
- GPT внешний — требует OpenAI API ключ в config.json
